MySQL和Oracle的区别

2020-04-06 Database MySQL, Oracle 0 评论

事务的提交

  • mysql默认自动提交事务
  • oracle默认手动提交事务，通过commit来提交

分页查询

  • mysql用limit进行分页
  • oracle用伪列rownum和嵌套查询

事务隔离级别

  • mysql默认repeatable read
  • oracle默认read committed

对事务的支持

  • mysql只有在InnoDB引擎时才支持事务
  • oracle任何情况都支持事务
